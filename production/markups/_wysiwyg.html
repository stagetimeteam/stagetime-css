.wysiwyg

This component is not like the others. It's for the content generated by users. So we allow them to use tags here like <p>, <ul>, <img> and so on.

This is the only component where we attach styles directly to tags. We can't ask users to use css classes so here we go with tags. But all the styles will work only in the context of .wysiwyg components and won't affect anything else on the site.

So basically by using this component in our layout we set up an area of the content that was written by users.

At this moment we only have two of this areas: organisation's description and content area of post. So not many tags have their unique look. Most of them below are in the default or draft condition.
