Size of image inside is based on 128px of height. Width can have any value but not smaller than 181px. Otherwise there is not enough space for dropdown menu.

Speaking about menu. There is .operations component inside. Please use it with .operations--for-media-video modifier specifically for this card.




.media-card--audio

1) First use .media-card--audio modifier

2) .operations component inside goes with .operations--for-media-audio modifier (for-media-audio, not for-media-video)

3) Background image, again, can have any size based on 128px but not smaller than 132x128. Wider -- ok.

4) Change icon to 16-wave

Generated background gradient based on user color. Format is:
background-image: linear-gradient(48.21deg, #BAC4B9 35.59%, rgba(186, 196, 185, 0) 100%);
