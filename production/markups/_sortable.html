.sortable

Let's use Sortable:
http://sortablejs.github.io/Sortable/

This is surprisingly well witten library, and it works just as is and covers all possible cases: it works with any tags, any styling, any margins around items, it works for both vertical and horizontal cases, and it works inside popups. No HTML or CSS adjustments required.

We need only Sortable.min.js file and it's initialization. In this project I initialize this plugin on .sortable class here: ../scripts/_sortable.js so just adding this class to any node in DOM is enough.

As for options I suggest to use:
animation: 150,
ghostClass: 'sortable-ghost'
so it looks better when sorting tiles with photo or something like that.
