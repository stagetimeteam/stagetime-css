.notification-center

This is wrapping component for set of .toast's. It adds positioning and scrollbar

Place .notification-center after modals.

First wrapper .notification-center can always be on the page. They won't overlap something. If there will be no toasts inside height will be 0px.

Once notification comes add it using jQuery's fade in.

It seems that we can close notification by clicking on it anywhere inside.

See real use case on the dashboard-notifications.html sample page
