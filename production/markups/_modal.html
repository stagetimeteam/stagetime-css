.modal

This is Bootstrap 3.4.1. modals with additional overriding styles in here: ../styles/_modal.css. White background and shadows were removed in the context of .modal--skeleton so this update won't hurt all original modals from legacy code. Vertical alignment was added to all modals including modals in legacy code.

CSS conflicts

Make sure to grab only modal related styles from bootstrap. Don't take built in things like normalize or helper classes and especially those code pieces that styles <a> and so on. This is, unfortunately, still happens if you use web ui generator and choose only modal checkbox. So styles have to be edited manually:

You can take this file ../vendors/bootstrap/bootstrap-modal-only.css



JS conflicts

When popup is open try to place .modal-open on <html> tag instead of <body>. This will prevent page jumping to the top on modal open. There is conflict with heigth: 100% that was set on <body> and necessary for layouting.
