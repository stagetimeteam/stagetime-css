The purpose of this component is just to create an image. We should think about it like if it was an image file. So it is not interactive. For hover effects, rounded corners and so on see .avatar




Size

To change size just use width and height attributes.
It will stretch image and keep its ratio.




Font size

In some extreme cases like super small 24x24 case we may want to increase font-size. Just change font-size property on text tag.
