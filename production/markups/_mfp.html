.mfp

For now I use jQuery plugin called Magnific Popup. .mfp is the name prefix they used in the code so all the things related to this popup component will have this prefix for naming.

This plugin is used only to create the functionality of popups: clicks, positioning, and so on. If something will go wrong we can replace it with another library or code our own. Only need to set up overlay color to rgba(51, 51, 51, 0.4);


Basic example

One minor thing to code. Add class .mfp-window wrapping each popup. It is needed fo aligning. So the init popup code will be: 

<div class="mfp-hide mfp-window" id="foo-bar">
    Lorem ipsum...
</div>

Click here to see default and unstyled popup

For this demo I created .mfp-handler and .mfp-cancel classes. Clicking on them will run and close popups. See see script sample here: ../scripts/_mfp.js

Later there will be galleries, YouTube videos, iframes and they will have their own types.
