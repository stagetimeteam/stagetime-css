Line breaks

By default we are not letting text in bubbles to break to new line. It is because inside of position: absolute line break will happen after each word.

So we assume default bubbles are meant to be used with short text like one above. If long text will appear inside of a bubble it will look like so:

Lorem ipsum dolor sit amet, consectetur adipiscing elit. Donec dolor tortor, tempor ac ipsum a, porttitor vestibulum orci. Fusce efficitur aliquet rutrum. Nulla massa lectus, fringilla ac orci a, efficitur luctus lacus.


br`s

In case of long text we can add line breaks manually, using <br>`s:

Lorem ipsum dolor sit amet, consectetur adipiscing elit.
Donec dolor tortor, tempor ac ipsum a, porttitor vestibulum orci.
Fusce efficitur aliquet rutrum. Nulla massa lectus,
fringilla ac orci a, efficitur luctus lacus.


.bubble--normal-wrap

Alternatively we can disable this no line breaks feature by adding .bubble--normal-wrap class and adding width manually. Like so style="width: 412px;":

Lorem ipsum dolor sit amet, consectetur adipiscing elit. Donec dolor tortor, tempor ac ipsum a, porttitor vestibulum orci. Fusce efficitur aliquet rutrum. Nulla massa lectus, fringilla ac orci a, efficitur luctus lacus.

Show/hide using .bubble-trigger

By default bubble is hidden using CSS. Basic way to show it is to place it into the context of .bubble-trigger class. Hovering this class (or tabbing inside) will shows the bubble.
