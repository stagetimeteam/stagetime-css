.modal-body

This is a single-div wrapper that set max-height and scrollbar inside of modal. Normally we would avoid making additional scrollbar inside of modal because it already has one somewhere in bootstrap but in some cases we want .modal-head and .modal-foot always be visible on the page. For example big form where save button always need to be visible.

For more details please see .modal
