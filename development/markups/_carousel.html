.carousel



Arrows via .action

We can use .action (with .action--faded modifier) inside of .carousel__control:





.carousel--arrows-on-top

This meant to be used with .action--transparent and 16-slim-left/16-slim-right icons





.carousel--faded-right

There will be few cases with arrows on the right. All of then need this fading gradient under.





.carousel--for-updates

In here we want very specific position for arrows (based on vertical center). and specific spacing between tiles.
