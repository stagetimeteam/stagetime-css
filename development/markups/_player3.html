.player3

Place it inside of .page, at the very end.

Please add class .has-player3 on the <html> tag if player is presented.

Clicking on .player3__handler collapses player. In this case please add .player3__handler on the <html>.

Please switch icons like so to make player work:
16-smaller-heart-outline to 16-smaller-heart
16-play to 16-pause
16-shuffle to 16-shuffled

.player3__action--disabled can be added to any action
