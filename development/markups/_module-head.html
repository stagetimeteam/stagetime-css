Note that layout didn't change despite 6px of height are missing

You can try to manually delete .module-head__right-slot from DOM in the develoepr tool and you'll see page won't shift up by 6px.





If we still need to use components that are taller than 40px we can adjust them using negative margins. For example these .action`s are 60px tall. So to compensate it we write something like: 

.module-head__right-slot .action {
    margin-top: -12px;
    margin-bottom: -8px;
}
