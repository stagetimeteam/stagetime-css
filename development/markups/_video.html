.video

Default video based just on width="" and height="". Pretty straightforward but doesn't make any sense in responsive layout.





Ratio

What we need to do is to let video to stretch and to fill parental element completely while keeping it's ratio. Unfortunately it is impossible via <iframe> or <video> tag. There are only two things in CSS that can keep ratio why width is changing: <img> tag and vertical padding.

Using padding-bottom we can set up element ratio to this component. So first you check upcoming video ratio and then choose one of following modifiers.

Change browser size to see how it works in different context.


.video--16x9


.video--4x3


.video--1x1




Any other ratio

Ratio can be set manually. Do calculation:
padding = height / width * 100

For example if we facing 21:9 vide calculations will be:
padding = 9 / 21 * 100 = 42.8571

Set this radio as: style="padding-bottom: 42.8571%;" to .video
Add these additional styles to .video__widget:
style="position: absolute; left: 0; top: 0; width: 100%; height: 100%;"
