.avatar

By default we assume that all of our photo and logos are rounded and clickable. We also provide a fancy hover effect here.




Size

Size of the image is controlled by width and height attributes. This component is flexible and will work with any image. Make sure to load correct files and set corresponding width and height attributes.




.avatar--squared

If we still need square instead of circle we have .avatar--squared modifier. Rounding of corners is set to 11px so it is not flexible in terms of image size. Smaller images may need less rounding. But this is unclear at this point what images of which sizes will be used. So for now we gonna live with default 11px:
