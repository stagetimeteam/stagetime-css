.parallax

See also .simple-parallax

In this default version scrollable area is itself and one screen before the image. So all the time when you see image or part of it is parallaxing. This works well for images deeper down to the page like if it was a landing page showing some features.

Unfortunately this method cuts a significant part of image. 300px by default. And this can be critical to people's photo. Also, half of this cut is never being seen if image is place on top of the page because there is no whitespace of window height above it. This is why on pages like profile page we gonna use alternative .simple-parallax component.

This .parallax component remains unused. Maybe until landing page or something. See demo:
