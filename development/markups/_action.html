.action--alternate

Width and height remain the same: 60x60.
Background ring changes to circle and becomes smaller.
Font size of text becomes smaller.
Colors are being changed.
It's all done by .action--alternate modifier.

In addition you may want to use icons that are more fatty in this case.

.action--multi-line

First we ease requirement for 60px height. Now height can be any. Each new line of text will add 12px to the height.
Secondly we increase spacing between icon and text.
This is being done by .action--multi-line modifier.

What's for text, please add <br> to make extra lines. No break requirements is still on to prevent accidental line breaks.
