.button--expandable

Hover it:

Customize

There is a problem with animation of the width. CSS can animate 32px to 108.094px but can't animate 32px to auto. There gotta be a value. And width value unknown in this case. It is based on text inside.

So what we have to do is to measure width of each .button--expandable and set it manually as: style="width: 108.094px;". After that animations will be able to work.

Note that you have to do in on window.load (not document.ready) because you need to wait while all fots are downloaded and applied.

See sample in ../scripts/_button.js
